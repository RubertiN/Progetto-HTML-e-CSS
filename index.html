Il linguaggio HTML consente di descrivere semanticamente
la struttura di un documento web attraverso tag.

CSS è un linguaggio usato per programmare la resa grafica
di documenti scritti in un linguaggio di markup, come HTML.

Bootstrap è una raccolta di strumenti liberi per la creazione di siti
e applicazioni per il web. Essa contiene modelli di progettazione
basati su HTML e CSS per le varie componenti dell'interfaccia.

SASS è un'estensione del linguaggio CSS che permette di utilizzare variabili,
di creare funzioni e di organizzare il foglio di stile in più file.
